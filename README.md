## Introduction

**SWAssistant** is a lightweight Chrome extension developed to interact with the browser-based game ShinobiWorld. This tool is created exclusively for educational purposes, allowing users to learn about browser automation, extension development, and web-based game interactions. This extension is not intended for competitive advantage or any other purpose beyond teaching.

---

## Features

- Automates interactions with the ShinobiWorld game interface.
- Demonstrates Chrome extension capabilities with JavaScript and DOM manipulation.
- Provides insights into event handling and UI enhancements in web games.

---

## Installation

1. Clone or download this repository:
   ```bash
   git clone https://github.com/SWAssistant1/SWAssistant.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** by toggling the switch in the top-right corner.
4. Click on the **Load unpacked** button.
5. Select the directory containing the SWAssistant source files.
6. The extension should now appear in the Chrome extensions bar.

---

## Usage

1. Navigate to the ShinobiWorld game website.
2. Ensure the SWAssistant extension is active in your Chrome browser.
3. Follow the in-extension prompts or features to interact with the game interface.
